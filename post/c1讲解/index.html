C

1、scanf纯数字输入

int n,m;
scanf("%d%d",&n,&m);

%d告诉scanf，要从缓冲区读一个数字，scanf会尝试从缓冲区中读入数字。

  • 如果遇到空格或回车，则会忽略该空格（同时缓冲区内也不会再有这个空格），直至遇到数字。
  • 如果遇到字母，则该%d读取赋值失败。同时该字母还会留在缓冲区。
  • scanf在赋值的同时也会有返回值，返回值为成功赋值的变量个数

2、常见错误

忘记回车、或者缺少空格。

#include <stdio.h>
int main()
{
    int n,m;
    scanf("%d%d",&n,&m);
    n-=((n-m)*2);
    printf("%d\n",n);
    if (n>=4)
        printf("Yes");
    else
        printf("No");
}
