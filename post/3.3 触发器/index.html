3.3 触发器

2023-11-12, 1309 words, 6 min read

一、触发器

1、定义

能够存储一位二值信号的基本单元电路

2、特点

  • 有两个稳定的状态：0状态和1状态；
  • 根据不同输入信号，置成0状态或1状态；
  • 当输入信号消失后，所置成的状态能够保持不变

3、分类

  • 按触发方式：电平触发、脉冲触发、边沿触发

    区分方法：

    电平触发

    脉冲触发 带有L符号

    边沿触发 clk带有三角，无圈为上升沿有效，有圈为下降沿有效

  • 按逻辑功能：SRSRSR、JKJKJK、TTT、DDD触发器

4、触发器电路结构和逻辑功能、触发方式的关系

  • 电路结构决定触发器的触发方式

  • 逻辑功能决定了稳态下的逻辑功能（输出逻辑）

    • 同一种逻辑功能的的触发器可以有不同的电路结构 实现，从而有不同的的触发方式（如边沿触发JK触发 器、电平触发JK触发器）

    • 同一种电路结构形式可以构成各种不同功能的触发器（如主从JK触发器、主从D触发器）

三、不同逻辑功能的触发器

1、分类

SRSRSR、JKJKJK、TTT、DDD触发器

2、逻辑函数

1、SR触发器：Qn+1=S+R′Qn(SR=0)Q^{n+1}=S+R'Q^n (SR=0)Qn+1=S+R′Qn(SR=0)

2、JK触发器：Qn+1=JQ′n+K′QnQ^{n+1}=JQ'^n+K'Q^nQn+1=JQ′n+K′Qn

3、T触发器：Qn+1=TQ′+T′QQ^{n+1}=TQ'+T'QQn+1=TQ′+T′Q

  • T=1时， 翻转
  • T=0时， 保持原态

4、D触发器：Qn+1=DQ^{n+1}=DQn+1=D

3、触发器的代换

  • J=S，R=K时， JK变成RS
  • J=K时， JK变成T

五、脉冲触发的触发器

1、脉冲触发的SR触发器（主从SR触发器）

①在CLK＝1CLK＝1CLK＝1时，主触发器按S、R变化，而从触发器保持状态不 变；

②在CLKCLKCLK由1 →0（下降沿），主触发器保持，从触发器随主触发器的状态翻转.故在CLKCLKCLK的一个周期内，触发器的输出状态 只可能改变一次

Qn+1=S+R′Qn(CP下降沿时)Q^{n+1}=S+R'Q^n (CP下降沿时)Qn+1=S+R′Qn(CP下降沿时)

2、主从JK触发器电路结构

Qn+1=JQ′n+K′Qn(CP下降时)Q^{n+1}=JQ'^n+K'Q^n (CP下降时)Qn+1=JQ′n+K′Qn(CP下降时)

CP下降沿
J K Qn+1Q^{n+1}Qn+1
0 0 QnQ^nQn 保持
0 1 0 置0
1 0 1 置1
1 1 Q′nQ'^nQ′n 翻转

3、特点

  • 在CLK＝1时，主触发器受输入信号控制，从触发器保持原态
  • 在CLK下降沿时，从触发器按主触发器状态翻转， 故触发器输出状态只能改变一次
  • 一次变化问题：在CLK=1期间，不论输入信号J、K变化多少次，主触发器能且仅能翻转一次，触发下降沿后，从触发器的输出只取决于最开始的那次翻转
