2、原码、反码、补码、移码

  • 原码： 符号位在前，0表示正数，1表示负数，数值位跟随符号位后面。

  • 反码： 正数的反码为其本身

    ​ 负数的反码通过其数值位逐位取反得到

  • 补码： 正数的补码为其本身

    ​ 负数的补码是其绝对值的原码取反加一。

  • **移码：**编码数的原码等于原数+2n−12^{n-1}2n−1

3、原码

1、方案：符号位在前，0表示正数，1表示负数，数值位跟随符号位后面。

2、缺陷

  • 存在2个零，即000020000_200002​和100021000_210002​ ，浪费有效编码
  • 编码数值不连续 ，硬件无法使用统一的加法 ，应希望数值随编码增大而增大

4、补码

1、方案（以5位二进制补码为例）

  • 最高位代表符号位：0～正，1～负
  • 10000代表最小的负数（-16），11111代表最大的负数（-1）

2、特点

  • 只有1个零
  • 符号确定后，数值随编码增长而增长

3、二进制补码的一般性表示

4、n位二进制补码表示的范围为−2n−1∼2n−1-2^{n-1}\sim 2^n-1−2n−1∼2n−1

5、补码体系的优势：只需要设计加法运算的硬件，减法 可以转换为加法运算，故不需为减法设计相应硬件

2、小于运算（有符号）

比较A<B流程：

  • A和B的补码均扩展1位符号位（防止减法溢出）
  • 执行减法，即C=A-B
  • 判断C的符号位
    • 如果为1，则C<0，即A<B为真
    • 如果为0，则C>=0，即A<B为假

3、小于运算（无符号）

比较A<B流程：

  • A和B的补码均扩展1位0（防止减法溢出）
  • 执行减法，即C=A-B
  • 判断C的符号位
    • 如果为1，则C<0，即A<B为真
    • 如果为0，则C>=0，即A<B为假

4、等于

方案一：采用XOR运算，然后判断结果是否全0

  • 执行XOR运算，即C=A⊕B
  • 将C的各位OR起来，得到1位结果
    • 如果为0，即“A=B”为真
    • 如果为1，即“A=B”为假

方案二：采用减法运算，然后判断结果是否全0

  • 执行减法运算，即C=A-B

  • 将C的各位OR起来，得到1位结果

    • 如果为0，即“A=B”为真

    • 如果为1，即“A=B”为假

状态机是数字系统中的核心部件，广泛应用于电路设计、FPGA程序设计以及软件设计等领域。根据不同的分类标准，状态机可以分为以下几类：

  1. 有限状态机 (FSM, Finite State Machine)：
    • 有限状态机指的是具有有限个状态的状态机。
    • 当接收到外部输入信号时，状态机会根据当前状态和输入信号综合考虑，从而跳转到下一个状态。
    • 状态机的三个关键点是：当前状态、外部输入、下一个状态。
  2. [Moore 型状态机](https://www.baidu.com/s?wd=Moore 型状态机&tn=25017023_2_dg&ch=8&usm=2&ie=utf-8&rsv_pq=f2adeb6000094455&oq=状态机分类&rsv_t=3965TJPNEFo%2FFDDZnIBq3IuVwE8gi6VwVCxB6IPNFoZhZLR6qm6z2UGwk9D7Jp72AgRD7g&sa=re_dqa_zy&icon=1)：
    • Moore 型状态机的输出只与当前状态有关，不受输入信号的影响。
    • 输入信号在这里主要起到激励作用，而不是直接改变状态机的输出。
  3. [Mealy 型状态机](https://www.baidu.com/s?wd=Mealy 型状态机&tn=25017023_2_dg&ch=8&usm=2&ie=utf-8&rsv_pq=f2adeb6000094455&oq=状态机分类&rsv_t=3965TJPNEFo%2FFDDZnIBq3IuVwE8gi6VwVCxB6IPNFoZhZLR6qm6z2UGwk9D7Jp72AgRD7g&sa=re_dqa_zy&icon=1)：
    • Mealy 型状态机的输出不仅与当前状态有关，还与输入信号有关。
    • 输入信号在这里既起到激励作用，也直接参与到状态机的输出中。

Moore型状态机（Moore Machine）和Mealy型状态机（Mealy Machine）的主要区别在于它们输出信号的产生方式。

  1. 输出只与当前状态有关：Moore型状态机的输出仅取决于当前的状态，而与输入信号的当前值无关。这意味着，如果状态机处于某个状态，并且输入保持不变，那么输出也将保持不变。Moore型状态机的输出信号是直接由状态寄存器译码得到的，因此它们属于同步输出状态机。123456

  2. 输出与当前状态和输入都有关：Mealy型状态机的输出不仅取决于当前状态，还与输入信号的当前值有关。这意味着，状态机的输出是输入信号和当前状态共同作用的结果。Mealy型状态机的输出信号是输入和状态变量的函数，因此它们属于异步输出状态机。

  3. 时序特性：Moore型状态机的输出在时钟边沿变化，总是在一个周期后发生。而Mealy型状态机的输出则可以随时发生变化，不依赖于时钟。在Mealy型状态机中，输入的更改可能会在逻辑完成后立即导致输出更改，这可能会导致异步反馈问题。78

  4. 安全性：Moore型状态机通常被认为更安全，因为它们的输出变化总是在时钟边沿之后，而Mealy型状态机则可能在逻辑完成后立即导致输出变化。

  5. 逻辑复杂度：在相同的周期内，Mealy型状态机对输入的反应更快，因为它们的输出函数是由输入和状态变量共同决定的，而Moore型状态机的输出函数只由状态变量决定。这可能导致Moore型状态机在时钟边沿之后需要更多的门延迟。
