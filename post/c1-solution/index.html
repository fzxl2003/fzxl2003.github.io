题目分析

输出如题所示的字符串即可，可以直接复制粘贴。

注意需要加上换行符，否则输出会挤在同一行。

错误示范

    printf("FoR rIFt ENergY uNnecEsSary iT iS?");

其运行结果为

FoR rIFt ENergY uNnecEsSary iT iS?oUtPut mAXimiZeD,BUg mAss pROducti0n...hEll0,WOrlD!?

    printf("FoR rIFt ENergY uNnecEsSary iT iS?\n");

#include <stdio.h>

B 无情的复读机

难度 考点
1 循环，输入输出

题目分析

根据题目要求，只需要读入一个正整数，然后输出对应次数的字符+即可。

观察数据范围 1≤n≤100001\leq n\leq 100001≤n≤10000 ，可知没有超过 int 类型变量能容纳的数值上限，使用 int 类型变量存储输入的数即可。输出指定次数的字符通常使用 for 或 while 循环，循环指定次数 nnn ，每次循环输出一次字符 + 且不换行，最后的输出结果就是连续的 nnn 个字符 + 。

示例代码

#include <stdio.h>

int main(){
    int i, n;	//声明变量
    scanf("%d", &n);	//读入输入数据
    for (i = 0; i < n; i = i + 1) {	//通过for循环输出指定次数
        printf("+");
    }
	
	return 0;
}

x=n−ma=m−x\begin{aligned} x&=n-m\\ a&=m-x \end{aligned} xa​=n−m=m−x​

扩展阅读：三目运算符

像逻辑非 ! 和负号 - 这种只连接一个数据的符号，称为“单目运算符”，例如 -a ；像加号 + 、减号 - 和乘号 * 这种连接两个数据的符号，称为“双目运算符”，例如 a+b ；以此类推，连接三个元素的符号就称为三目运算符。

在C语言中，只有一种三目运算符，一般形式为： 表达式1 ? 表达式2 ：表达式3

在运算时，程序会先求解 表达式1 ，若其值为真（非 000 ）则将 表达式2 的值作为整个表达式的取值，否则（ 表达式1 的值为 000 ）将 表达式3 的值作为整个表达式的取值。这里的“表达式”，既可以是数，也可以是字符串。

例如，语句 int Max = (a > b) ? a : b; 的意思就是：如果 a>ba>ba>b ，那么 Max=aMax=aMax=a ，否则 Max=bMax=bMax=b

我们可以利用三目运算符来缩短本题的代码：

#include <stdio.h>

int main() {
    int m, n;
    scanf("%d%d", &n, &m);
    int x = n - m;//没去的次数
    int a = m - x;
    printf("%d\n",a);
	printf(a >= 4 ? "Yes" : "No");
    return 0;
}

注意以下几点

  • 题目为多组数据输入 需要使用循环结构进行输入输出。
  • 题目所要求计算的是 b  mod  ab\ \bmod\ ab mod a 而非 a  mod  ba\ \bmod\ ba mod b ，不要搞错了变量。
  • 在进行取余运算之前就要先行判断 aaa 是否为 000 再做运算，否则程序会在试图对 000 取余的时候崩溃。

#include <stdio.h>
int main(void) {

        scanf("%d%d", &a, &b);
        if (a == 0) {
            printf("Wait. That's illegal.\n");
        } else {
            printf("%d mod %d = %d\n", b, a, b % a);
            // 注意题干 计算的是 b mod a 而不是 a mod b
        }
    }
    return 0;
}

F 军乐团选拔

难度 考点
2 四则运算 条件判断

题目分析

本题实际上为计算 nnn 个自然数中所有不为 000 的数的平均数，应该用一个变量 num 存储所有不为 000 的数的总和，再用另一个变量 cnt 记录人数不为 000 的连队数目。
具体实现方式为每读入一个连队的人数 a，就进行一次判断，若 a 不为 000 则将不为零的连队数 +1+1+1。

if(a != 0)
{
    num += a;  //更新总人数num
    cnt++;  //更新参与计算平均人数的连队数目
}

最后只需判断 cntcntcnt 是否为 000，即可确定输出平均数还是 Oh!No! 了。

示例代码

#include<stdio.h>

int main()
{
    int n, a, ans, num=0, cnt=0; //num和cnt是不断累加的，所以一定要初始化成0
    scanf("%d", &n);
    for(int i = 1; i <= n; i++)
    {
        scanf("%d", &a);
        if(a != 0)
        {
            num += a;  //更新总人数num
            cnt++;  //更新参与计算平均人数的连队数目
        }
    }

    if(cnt != 0)  //有人报名
    {
        ans = num / cnt;  //平均人数
        printf("%d", ans);
    }
    else  //无人报名
        printf("Oh!No!");
    
    return 0;
}

H 永久的爱

难度 考点
3 整数除法，抽屉原理

题目分析

先考虑求最大值，这是我们将所有的 111 全部排在一起，最大值显然为 bbb 。
接下来考虑求最小值，问题可以描述为用 a−ba-ba−b 个 000 去分隔 bbb 个 111 ，相当于将 bbb 个 111 放到 a−b+1a-b+1a−b+1 个抽屉里，由抽屉原理可知，至少有一个抽屉中的 111 的数量不小于 $\left \lceil \frac{b}{a-b+1} \right \rceil $ （向上取整），这就是最小值。

由 ⌈xy⌉=⌊x+y−1y⌋\left \lceil \frac{x}{y} \right \rceil = \left \lfloor \frac{x+y-1}{y}\right \rfloor⌈yx​⌉=⌊yx+y−1​⌋ （向下取整），知 ⌈ba−b+1⌉=⌊aa−b+1⌋\left \lceil \frac{b}{a-b+1} \right \rceil = \left \lfloor \frac{a}{a-b+1}\right \rfloor⌈a−b+1b​⌉=⌊a−b+1a​⌋ ，可以用表达式 a / (a - b + 1) 表示。

示例代码

#include <stdio.h>
int main()
{
    int t;
    scanf("%d", &t);
    while(t--)
    {
        int a, b;
        scanf("%d%d", &a, &b);
        printf("%d %d\n", b, a / (a - b + 1));
    }
    return 0;
}

I 军乐团分组

难度 考点
4 最大公因数 gcdgcdgcd

题目分析

分析题意不难发现，分组要求同组人必须来自同连，也就是说每组人数必须是所有连人数的公因数。但不要被样例解释部分误导，不需要求每一连队分了多少组，只需要用 总人数 /// 每组人数即可求出总组数 。因此用一个 num 变量记录 888 个连的总人数，再用一个变量 gcd 循环求取 888 个连人数的最大公因数 (gcd)(gcd)(gcd)。

示例代码

#include<stdio.h>

int main()
{
    int n, a, b, gcd, num = 0;
    scanf("%d", &a);
    gcd = a;
    num = a;
    for(int i = 2; i <= 8; i++)
    {
        scanf("%d", &b);
        num += b;
        
        if(gcd > b)  //求最大公因数
            gcd = b;
        while(!(a % gcd == 0 && b % gcd == 0))
        {
            gcd--;
        }
        a = gcd;
    }
    n = num / gcd;
    printf("%d", n);
    return 0;
}

int a,b,gcd,temp;  //求a和b的最大公因数
scanf("%d%d",&a,&b);
if(a < b)
{
    temp = a;
    a = b;
    b = temp; //交换a和b的值
} //始终保证a是二者中较大的数

    temp = a % b;
    a = b;

}

J 分割平面

难度 考点
5~6 计数，取模

本题难度过高，不做要求。

题目解析

若使得区域最多，显然需要使尽量多的直线不共点，且任意两条直线不相互平行。即除了初始的 nnn 个点以外，任意点不能同时在三条直线上，且任意两条直线都有交点。

由 Hint\text{Hint}Hint 中的欧拉示性数公式 v−e+f=1v-e+f=1v−e+f=1 ，分割的区域数量就是公式中的 fff ，因此我们需要求得 v,ev,ev,e 的值，即可通过 f=e−v+1f=e-v+1f=e−v+1 求得答案。

如何求顶点数 vvv ？

顶点有两个来源，第一种是最初给出的 nnn 个点，第二种是若干条直线的两两相交新增的交点。

由两点确定一条直线，一个新增交点涉及到两条直线， 444 个初始点。

然而 444 条初始点能够确定 333 对相交的直线，如下图：

![题解图1](D:\Student Works\Academic Assistance\2023程序设计基础助教\出题\分割平面\题解图1.jpg)

即任选 444 个初始点会产生 333 个新增交点，因此产生的新增交点总数为 3⋅Cn43\cdot C_n^43⋅Cn4​ 。

所以顶点总数 v=n+3⋅Cn4v=n+3\cdot C_n^4v=n+3⋅Cn4​ 。

如何求边数 eee ?

两点确定一条直线，对于每一条直线，若不计新增交点，该直线被两个初始点分成了 333 段，即每条直线最初都贡献了 333 条边，一共有 Cn2C_n^2Cn2​ 条直线，所有直线最初贡献了 3⋅Cn23\cdot C_n^23⋅Cn2​ 条边。

对于一条直线，每个新增交点会增加一条边。而每个新增交点会作用于两条直线，即每个新增交点贡献了 222 条边，因此所有新增交点贡献了 2⋅3⋅Cn4=6⋅Cn42\cdot 3\cdot C_n^4 = 6\cdot C_n^42⋅3⋅Cn4​=6⋅Cn4​ 条边。

所以边的总数 e=3⋅Cn2+6⋅Cn4e = 3\cdot C_n^2+6\cdot C_n^4e=3⋅Cn2​+6⋅Cn4​ 。

因此最终答案为 f=e−v+1=3⋅Cn2+3⋅Cn4−n+1=3n(n−1)2+n(n−1)(n−2)(n−3)8−n+1f = e - v + 1 = 3\cdot C_n^2 + 3\cdot C_n^4 - n + 1 = \frac{3n(n-1)}{2} + \frac{n(n-1)(n-2)(n-3)}{8} - n + 1f=e−v+1=3⋅Cn2​+3⋅Cn4​−n+1=23n(n−1)​+8n(n−1)(n−2)(n−3)​−n+1 。

注意事项：

  1. 需要用 long long 类型
  2. 注意模运算和数据范围，记得在运算过程中取模，具体看代码。

示例代码

#include <stdio.h>
int main()
{
	long long n;
	long long mod = 998244353; //模数
	long long inv2 = (mod + 1) / 2; //除以2等价于乘inv2

第二种解法 by cwz

首先考虑这个问题：对于平面上 mmm 条直线，任意三线不共点，最多能够把平面划分成多少个区域？

111 条直线最多可以将平面划分为 222 个区域，222 条直线最多可以将平面划分为 444 个部分，333 条直线最多可以将平面划分为 777 个部分。每增加一条直线，都最多可以和平面上已有的 i−1i - 1i−1 条直线相交，增加 iii 个区域。由此，我们可以得到，对于平面上 mmm 条直线，任意三线不共点，最多能够把平面划分成 m(m+1)2+1\frac{m(m+1)}{2} + 12m(m+1)​+1 个区域。

我们已经知道了这个题有 n(n−1)2\frac{n(n-1)}{2}2n(n−1)​ 条直线。可是，对于这个题来说，nnn 个点确定 n(n−1)2\frac{n(n-1)}{2}2n(n−1)​ 条直线时，显然组成的图形会有 nnn 个点不是只由两条直线相交得到的，而是 n−1n - 1n−1 条直线的公共点。

所以接下来考虑第二个问题：当 kkk 条直线汇聚于一点时，会比这 kkk 条直线任意三线不共点时把平面少划分多少区域？

显然当 kkk 条直线汇聚于一点时，可以把平面划分为 2k2k2k 个区域，k(k+1)2+1−2k=k2−3k+22=(k−1)(k−2)2\frac{k(k+1)}{2} + 1 - 2k = \frac{k^2 - 3k + 2}{2}=\frac{(k-1)(k-2)}{2}2k(k+1)​+1−2k=2k2−3k+2​=2(k−1)(k−2)​。因此，当 kkk 条直线汇聚于一点时，会比这 kkk 条直线任意三线不共点时把平面少划分 (k−1)(k−2)2\frac{(k-1)(k-2)}{2}2(k−1)(k−2)​ 个区域。

综合以上问题的考虑，对于这个题，直线数量 m=n(n−1)2m = \frac{n(n-1)}{2}m=2n(n−1)​，不是只由两条直线相交得到的点有 nnn 个，这 nnn 个点全部都是 n−1n - 1n−1 条直线的公共点，即 k=n−1k = n - 1k=n−1，我们可以写出平面区域数 fff 的最终表达式：

f=n(n−1)2⋅(n(n−1)2+1)2+1−n⋅(n−3)(n−2)2f = \cfrac{\cfrac{n(n-1)}{2}\cdot\left(\cfrac{n(n-1)}{2} + 1\right)}{2} + 1 - n \cdot \cfrac{(n-3)(n-2)}{2} f=22n(n−1)​⋅(2n(n−1)​+1)​+1−n⋅2(n−3)(n−2)​
