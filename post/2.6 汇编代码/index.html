二、数据区域 .data

1、功能

存储程序中全局变量的一块内存区域

空间大小在程序运行前就已确定

可读可写

2、格式

<lable>:.<type> <space/初值>

<lable> 变量名

<type> 空间类型

3、空间类型

type 基本用途 数组形式 初始化形式
.byte 声明8位变量 a:.byte 0:127 a:.byte 4
.half 声明16位变量 同上 同上
.word 声明32位变量 同上 同上
.ascii 声明字符串，结尾无'\0' 不可定义空间大小 a:.ascii "123"
.asciiz 声明字符串，结尾有'\0' a:.asciiz "123"
.space 为一个变量预留指定的字节数 array:.space 100
  • 汇编语言关注的是变量的存储类型：一个变量需要占用多少字节
  • 汇编语言的变量类型都是无符号的

三、代码区域 .text

1、功能

存放程序中指令序列的一块内存区域

空间大小在程序运行前就已确定

只读（某些CPU允许代码段为可写）

2、MARS模拟的相关指令 syscall

1、操作步骤

  • 在寄存器 $v0 中加载服务号。
  • 按照指定在 $a0、$a1、$a2或$f12中加载参数值（如果有）。
  • 发出SYSCALL指令。
  • 按照指定从结果寄存器中检索返回值（如果有）。

2、例子

#显示存储在t0中的int数字 
li  $v0, 1           # service 1 is print integer
add $a0, $t0, $zero  
syscall

3、表格

功能 $v0 编码 参数 结果
打印int变量 1 $a0 = integer to print
打印float变量 2 $f12 = float to print
打印double变量 3 $f12 = double to print
打印字符串 4 $a0 = address of null-terminated string to print
打印字符 11 $a0 = character to print
读int变量 5 $v0 contains integer read
读float变量 6 $f0 contains float read
读double变量 7 $f0 contains double read
读字符串 8
读字符 12 $v0 contains character read
退出程序 10
