3.3 天球坐标系

2023-10-28, 1205 words, 5 min read

一、天球

1、定义

假想的一个球，以任意指定点为球心（地心、观测者的眼睛、日心、太阳系质心……）任意长为半径 。通常，天球的半径取单位长度 1

2、天体的球面坐标定义

按照中心投影方式把天体投影到天球球面上，在天球上规定基本大圆和起始原点建立天球坐标系，从而建立天体的球面坐标。

  • 天体投影在天球上的二维球面角度坐标反映的只是天体的方位，并不反映出天体到天球球心的距离。

3、天球上的基本点、线、圆

1、天轴：将地球自转轴无限延长与天球相交所得的直径PNPSP_NP_SPN​PS​，称为天轴。

2、天极：天轴与天球球面相交二点PSP_SPS​和PNP_NPN​，和地球北极对应的PNP_NPN​点称为天北极，另一点PSP_SPS​称为天南极。

3、天赤道：与天轴垂直的大圆称为天赤道QQ′ 。

5、天顶：观测者所在地点A的铅垂线的反向延长线与天球的交点 Z 称为天顶

6、天底：观测者所在地点与地心连线的延长线与天球的交点Z ′称为天底。

7、真地平圈：与天顶、天底ZZ ′垂直的大圆ESWN称为真地平圈。

8、天子午圈、四方点和卯酉圈

天顶、天底、天子午圈、卯酉圈、四方点和地平圈都不参与天球的视运动。

  • 天子午圈：通过天顶Z和天极P的大圆是天子午圈，天子午圈有地方性。

  • 卯酉圈：过天顶而与天子午圈垂直的大圆是卯酉圈，可见，卯酉圈也有地方性。

  • 四方点（东西南北点）：

    • 天子午圈和地平圈相交于S和N点，靠天北极PN的那点N为北点，另一S点为南点

    • 卯酉圈和地平圈相交于E和W点，观测者面向北点，右方E点称为东点，左方W点称为西点。

9、黄道和黄极

  • 黄道：地球公转轨道投影到日心天球上的大圆，就是黄道，该平面称为黄道面。
  • 黄极：黄道有两个极，靠近天北极P的极称为北黄极K，另一极称为南黄极K′ 。
  • 黄赤交角：黄道和赤道的交角ε = 23°26′21″.488 。

10、分至点

  • 二分点：太阳周年视运动，自南向北穿越赤道的交点是春分点Υ ，自北向南穿越赤道的交点是秋分点。
  • 二至点：黄道上到二分点相距90°的两点为二至点，赤道以北的叫夏至点，赤道以南的叫冬至点。

4、视运动

  1. 天体的周日视运动：由于地球自转的原因，人们感受到天球绕天轴反方向旋转，一日一周，造成天体的周日视运动。
  2. 太阳的周年视运动：地球绕太阳公转，对于地球上的观测者来讲，太阳在一年内沿着黄道自西向东（从北黄极K看逆时针方向）转动一周。
  3. 上帝视角下，地球自转和公转均为逆时针。

5、约定方向

  • 地图的约定方向：观测者在地球外时，上北下南，左西右东。
  • 天图的约定方向：观测者只能由内向外看天球上北下南，左东右西
