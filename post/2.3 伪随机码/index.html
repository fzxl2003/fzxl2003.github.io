一、伪随机码

1、随机码

1、定义：具有良好的自相关性的非周期序列。

2、特点：自相关性、非周期、不遵循任何编码规则、实际中无法复制与利用。

2、伪随机码

1、定义：具有随机码序列良好相关特性的有周期性的序列

2、特点：良好的自相关性、具有某种确定的编码规则、周期性、容易复制。

3、二进制伪随机码：由“0”和“1”组成的伪随机序列，简称伪码。

二、二进制序列的计算

1、二进符号序列、二进信号波形

1、二进符号序列 $\left { x \right } $ ：用 1 和 0 表述

2、反馈逻辑

1、定义：根据当前寄存器状态经过计算输出至第一个寄存器的计算法则。

3、线性移位寄存器输出特性

1、输出序列是一个周期序列

2、当初始状态是0状态时，线性移位寄存器的输出是个0序列。

3、级数相同的线性移位寄存器的输出序列与寄存器的反馈逻辑有关。
