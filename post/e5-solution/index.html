A 无幂之幂

考点 难度
函数 1

题目分析

如题目描述，由于禁止在代码中出现 math.h , pow , abs 这三个字符串，因此我们不调用 math.h 库中的 pow 函数和 fabs 函数，以及 stdlib.h 中的 abs 函数，而是自己声明函数用于计算。注意不仅是不能调用这几个函数，代码别的地方同样不能含有以上内容(例如不能声明函数名为 Mypow ，不能声明变量名为 power)。

int myPow(int a, int b) {
    int ans = 1;
    for (int i = 0; i < b; i++) {
        ans = ans * a;
    }
    return ans;
}
int myAbs(int a) {
    if (a >= 0) {
        return a;
    } else {
        return -a;
    }
}

如果使用三目运算符和宏定义的话，myAbs 也可以写成如下形式

#define myAbs(a) ((a) > 0 ? (a) : (-(a)))

注意这些括号的使用，因为使用宏定义本质上是进行文本的替换，若不加上括号那么替换后的文本的运算顺序可能与你预期不相符，例如可以试试以下代码

#include <stdio.h>
#define myAbs(a) a > 0 ? a : -a
int main(void) {
    printf("%d", myAbs(1 - 3));
    return 0;
}

你会发现运行结果为 -4

在这个代码中，将myAbs(1 - 3)进行文本替换后得到的结果是

1 - 3 > 0 ? 1 - 3 : - 1 - 3

也即

-2 > 0 ? -2 : -4

这显然与我们所期望的效果不同，因此需要加上括号来矫正运算顺序。

如果真的一定要使用库函数的话，也不是不可以，绕过字符串匹配的方法有很多，例如使用标记粘贴运算符

#define A(x) fab##s(x)
#define P(a,b) po##w(a,b)

以及使用宏延续运算符

#define H <math\
.h>

w
#define A fab\
s

示例代码

#define myAbs(a) ((a) > 0 ? (a) : (-(a)))
int myPow(int a, int b) {
    int ans = 1;
    for (int i = 0; i < b; i++) {
        ans = ans * a;
    }
    return ans;

int main(void) {
    int a, b;

    }

}

D - 经典的三角形

难度 考点
3 函数

题目分析

首先筛选出最大的边，将其换到 aaa 边。然后利用三角形 a2+b2a^2+b^2a2+b2 与 c2c^2c2 的关系，判断三角形最大角的形状，接着再用三条边的长度判断等腰性。

将以上思路封装成一个函数，按题目要求的顺序调用即可。

示例代码

#include <stdio.h>
void triangle(int i, int a, int b, int c) {
    printf("Question %d:\n", i);
    if(b > a && b > c) {
        int t = a; a = b, b = t;
    }
    if(c > a && c > b) {
        int t = a; a = c, c = t;
    }

    if(b + c <= a) {
        printf("no triangle\n");
        return ;
    }
    
    if(b * b + c * c > a * a)
        printf("acute triangle\n");
    else if(b * b + c * c == a * a)
        printf("right triangle\n");
    else printf("obtuse triangle\n");

    if(a == b && a == c)
        printf("equilateral triangle\n");
    else if(b == c || a == b || a == c)
        printf("isosceles triangle\n");
}
int main() {
    int a[15];
    for(int i = 1; i <= 10; i ++)
        scanf("%d", &a[i]);
    triangle(1, a[1], a[5], a[9]);
    triangle(2, a[2], a[7], a[10]);
    triangle(3, a[2], a[3], a[6]);
    triangle(4, a[8], a[9], a[10]);
    triangle(5, a[6], a[7], a[10]);
    triangle(6, a[3], a[4], a[8]);
    triangle(7, a[1], a[2], a[6]);
    return 0;
}

E 计算不确定度

难度 考点
3 循环，浮点数，计算

题意分析

很建议做一份计算不确定度的代码保存起来，能用excel之类的做就更好了，做基物实验的时候真的用得上

根据题意进行计算。注意要将所有数据读入后计算得到的 avgavgavg 才是正确的 avgavgavg

示例代码

#include <math.h>
#include <stdio.h>
int main(void) {
    double data[1000], avg = 0, u = 0;
    int n = 0;
    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        scanf("%lf", &data[i]);
        avg += data[i];
    }
    avg /= n;
    for (int i = 0; i < n; i++) {
        u += (data[i] - avg) * (data[i] - avg);
    }
    u = sqrt(u / n / (n - 1));
    printf("%.6f\n%.6f", avg, u);
    return 0;
}

F 哪吒的水题

难度 考点
4 二分法、三分法

题目分析

设供水站横坐标为 xxx ，则供水站到 SimonSimonSimon 家 (x1,y1)(x_1, y_1)(x1​,y1​) 的距离为 (x−x1)2+y12\sqrt {(x-x_1)^2+y_1^2}(x−x1​)2+y12​​ ，到哪吒家 (x2,y2)(x_2, y_2)(x2​,y2​) 距离为 (x−x2)2+y22\sqrt {(x-x_2)^2+y_2^2}(x−x2​)2+y22​​ ，因此总费用 s(x)=s1(x−x1)2+y12+s2(x−x2)2+y22s(x) = s_1\sqrt {(x-x_1)^2+y_1^2}+s_2\sqrt{(x-x_2)^2+y_2^2}s(x)=s1​(x−x1​)2+y12​​+s2​(x−x2​)2+y22​​ .

显然，供水站最佳位置的横坐标在区间 [x1,x2][x_1, x_2][x1​,x2​] 中，我们要在该区间内找到一个合适的 xxx 使得 s(x)s(x)s(x) 最小，并输出此时的 xxx 和 s(x)s(x)s(x) .

法一：二分法

求函数极值点可以使用求导函数零点的方法。

s(x)s(x)s(x) 是关于 xxx 的先减后增函数， s′(x)=s1x−x1(x−x1)2+y12+s2x−x2(x−x2)2+y22s'(x) = s_1 \frac {x-x_1}{\sqrt {(x-x_1)^2+y_1^2}}+s_2 \frac {x-x_2}{\sqrt {(x-x_2)^2+y_2^2}}s′(x)=s1​(x−x1​)2+y12​​x−x1​​+s2​(x−x2​)2+y22​​x−x2​​ ，一般情况下先负后正，在区间 [x1,x2][x_1, x_2][x1​,x2​] 上只有一个零点，可以用二分法求该导函数零点。

示例代码

#include <stdio.h>
#include <math.h>
//math.h库中有个函数名字叫y1，如果声明全局变量的话不能被命名为y1，奇怪的事情发生了
#define eps (1e-8)
int main()
{
	double a, b, c, d, s1, s2; //a, b, c, d分别为x1, y1, x2, y2
	scanf("%lf%lf%lf%lf%lf%lf", &a, &b, &c, &d, &s1, &s2);
	double l = a, r = c, mid = (l + r) / 2;
    double dis1, dis2, f, s; //分别记录距离1，距离2，s'(x)的值，s(x)的值
	while(r - l > eps) {
        dis1 = sqrt((mid - a) * (mid - a) + b * b);
        dis2 = sqrt((mid - c) * (mid - c) + d * d);
        f = s1 * (mid - a) / dis1 + s2 * (mid - c) / dis2; //计算s'(mid)
		if(f > 0) //若导函数s'(mid)>0
			r = mid; //此时供水站只可能在区间[l, mid]上，更新r = mid
		else
			l = mid; //此时供水站只可能在区间[mid, r]上，更新l = mid
		mid = (l + r) / 2; //更新mid
	}
    //此时l, mid, r之间的差小于eps，一定精度范围下可视为相等，均为满足要求的供水站横坐标
    dis1 = sqrt((mid - a) * (mid - a) + b * b);
    dis2 = sqrt((mid - c) * (mid - c) + d * d);
    s = s1 * dis1 + s2 * dis2; //计算s(mid)
	printf("%.3f %.3f", mid, s); //输出mid, s(mid)
	return 0;
}

法二：三分法

易知 s(x)s(x)s(x) 一般情况下是关于 xxx 的先减后增函数，可以选择使用三分法求该函数极小值点。

三分法类似二分法，可以用于求单峰函数的极值点，下面以本题为例对其思路做简单说明。

设极小值点为 (x0,s(x0))(x_0, s(x_0))(x0​,s(x0​)) . 初始化 l=x1,r=x2l=x_1, r=x_2l=x1​,r=x2​ ，每次循环，在区间 [l,r][l, r][l,r] 中选择两个点 mid1<mid2mid1<mid2mid1<mid2 ，如区间 [l,r][l, r][l,r] 的三等分点 mid1=2l+r3,mid2=l+2r3mid1=\frac {2l+r} 3, mid2=\frac {l+2r} 3mid1=32l+r​,mid2=3l+2r​ 。如果 s(mid1)>s(mid2)s(mid1)>s(mid2)s(mid1)>s(mid2) 则说明 x0x_0x0​ 在区间 [mid1,r][mid1, r][mid1,r] 中（若在 [l,mid1)[l, mid1)[l,mid1) 中，则说明 s(x)s(x)s(x) 在 [x0,r][x_0, r][x0​,r] 上单调递增，与 s(mid1)>s(mid2)s(mid1)>s(mid2)s(mid1)>s(mid2) 矛盾），更新 l=mid1l=mid1l=mid1 ；否则说明 x0x_0x0​ 在区间 [l,mid2][l, mid2][l,mid2] 中，更新 r=mid2r=mid2r=mid2 。每次循环缩小区间长度为原来的 23\frac 2332​ ，当 r−lr-lr−l 缩小到足够小时，即可得到满足精度要求的答案。

示例代码

#include <stdio.h>
#include <math.h>
#define eps (1e-8)
//s函数用到了很多次，利用宏定义简化代码，在之后学了函数后可以将s(x)定义为函数
#define s(x) (s1 * sqrt((x - a) * (x - a) + b * b) + s2 * sqrt((x - c) * (x - c) + d * d))
int main()
{
	double a, b, c, d, s1, s2;
	scanf("%lf%lf%lf%lf%lf%lf", &a, &b, &c, &d, &s1, &s2);
	double l = a, r = c, mid1 = (2 * l + r) / 3, mid2 = (l + 2 * r) / 3;
	while(r - l > eps) {
		if(s(mid1) > s(mid2)) l = mid1; //此时供水站只可能在区间[mid1, r]上，更新l=mid1
		else r = mid2; //此时供水站只可能在区间[l, mid2]上，更新r=mid2
		mid1 = (2 * l + r) / 3; //更新mid1
		mid2 = (l + 2 * r) / 3; //更新mid2
	}
    //此时l, mid1, mid2, r之间差小于eps，一定精度范围下可视为相等，均为满足要求的供水站横坐标
	printf("%.3f %.3f", mid1, s(mid1)); //输出mid1, s(mid1)
	return 0;
}

G 格雷码2023

难度 考点
5 递归 分治

题目分析

这道题采用了一种分治递归的思想，在求解 nnn 位格雷码的问题划分成求解 n−1n − 1n−1 位格雷码的问题，以此类推直至划分成求解111 位格雷码的问题。

可以用递归的思路实现。具体而言，设递归函数 f(n,x)f(n,x)f(n,x) 的功能是输出 nnn 位格雷码的第 xxx 个， xxx 从 000 开始记，如下图， f(3,4)f(3,4)f(3,4) 就是输出 110 。

对于 f(k,x)f(k,x)f(k,x) ，我们可以先输出最高位，然后问题转化为输出 k−1k-1k−1 位的格雷码。如果第 xxx 个格雷码在 2k2^k2k 个格雷码中的前一半，即 x<2k−1x<2^{k-1}x<2k−1 ，则最高位为 000 ，后面的部分等价于 k−1k-1k−1 位格雷码的第 xxx 个，因此递归调用 f(k−1,x)f(k-1,x)f(k−1,x) 即可；如果第 xxx 个格雷码在 2k2^k2k 个格雷码中的后一半，即 x≥2k−1x\ge 2^{k-1}x≥2k−1 ，则最高位为 111 ，观察上方图片 k=3k=3k=3 的例子，根据对称关系，可知后面的部分等价于 k−1k-1k−1 位格雷码的第 2k−1−x2^k-1-x2k−1−x 个，因此递归调用 f(k−1,2k−1−x)f(k-1,2^k-1-x)f(k−1,2k−1−x) 即可。

递归的基本情况为 k=0k=0k=0 时，即所谓 000 位格雷码，应结束递归，不输出内容。由于本题中每输出一个格雷码之后都要换行，可以把输出换行写在递归的基本情况中，即基本情况 f(0,0)f(0,0)f(0,0) 输出一个换行符。

最后在主函数中输出全部格雷码，即从第 000 个一直输出到第 2n−12^n-12n−1 个 nnn 位格雷码，从 f(n,0)f(n,0)f(n,0) 一直调用到 f(n,2n−1)f(n, 2^n-1)f(n,2n−1) 即可。

示例代码 1 - 递归

#include <stdio.h>

void f(int k, int x)
{
	if(k == 0)
		printf("\n");
	else if(x < (1 << (k - 1)))
	{
		printf("0");
		f(k - 1, x);
	}
	else
	{
		printf("1");
		f(k - 1, (1 << k) - 1 - x);
	}
}

int main()
{
	int n;
	scanf("%d", &n);
	for(int i = 0; i < 1 << n; ++i)
		f(n, i);
	return 0;
}

示例代码 2 - 二维数组

二维数组的做法，可以参考。

111 位格雷码为 000 和 111 ，利用通过 111 位格雷码构造出 222 位格雷码，再通过 222 位格雷码构造出 333 位格雷码。直到构造出 nnn 位格雷码。

#include <stdio.h>
int a[2048][2048];
//求n位格雷码
void GrayCode(int n, int num)
{
    if (n == 1) //分治到只有1位Gray码的情况
    {
        a[0][0] = 0;
        a[1][0] = 1;
        return;
    }
    //将求解n位Gray码的问题划分成求解n-1位Gray码的问题
    GrayCode(n - 1, num / 2);
    //n位Gray码的前半部分最高位置0，其余位不变
    for (int i = 0; i < num / 2; ++i)
    {
        a[i][n-1] = 0;
    }
    //n位Gray码的后半部分最高位置1，其余位由n-1位Gray码翻转而来
    for (int i = num / 2; i < num; ++i)
    {
        a[i][n-1] = 1;
        for (int j = 0; j < n - 1; ++j)
        {
            a[i][j] = a[num - i - 1][j];
        }
    }
}
int main()
{
    int n;
    scanf("%d",&n);
    int num = 1<<n; //n位Gray码的个数num
    GrayCode(n, num);//求num个n位Gray码
    //输出num个n位Gray码
    for (int i = 0; i < num; ++i)
    {
        for (int j = n - 1; j >= 0; --j)
        {
            printf("%d",a[i][j]);
        }
        printf("\n");
    }
    return 0;
}

由 Gino 摆放能量石的限制规则很容易让人联想到 C5 的 H 题，其实本质上这题就是对于 “男厕尴尬定理” “禁止抄袭！”的二维拓展。当无法继续放入能量石之后，区域内的所有能量石会构成一个 a×ba \times ba×b 的方阵，而 aaa 和 bbb 分别对应的就是 C5 的 H 题中座位总数为 mmm 和 nnn 时可以容纳的学生总数。在此再次把两题的灵感来源放在这里 (ಡωಡ)

【毕导】男同胞福音！如何解决尿尿时最尴尬的难题？建议偷偷收藏

当然，如果你想使用数组来模拟这个过程的话肯定是不现实的，mmm 和 nnn 可能很大，每放一块能量石计一次数的效率过于低下。此题不必纠结能量石的摆法，只需要求能量石的数量。在此给出两种方法。

方法一：递归法

如果总数 nnn 为奇数，那么第一个能量石放在区域 111，第二个能量石放在区域 nnn，第三个能量石放在区域 n+12\cfrac{n + 1}{2}2n+1​。左边 n+12\cfrac{n + 1}{2}2n+1​ 个区域可以放 f(n+12)f(\cfrac{n + 1}{2})f(2n+1​) 块能量石，右边 n+12\cfrac{n + 1}{2}2n+1​ 个区域也可以放 f(n+12)f(\cfrac{n + 1}{2})f(2n+1​) 块能量石，那么去掉中间那个被重复计算的能量石之后，此时 nnn 个区域可以放置的能量石个数为 $$f(n) = 2 f(\cfrac{n + 1}{2}) - 1$$

如果总数 nnn 为偶数，那么第一个能量石放在区域 111，第二个能量石放在区域 nnn，第三个能量石放在区域 n2\cfrac{n}{2}2n​（中间两个位置的效果一样）。左边 n2\cfrac{n}{2}2n​ 个区域可以放 f(n2)f(\cfrac{n}{2})f(2n​) 块能量石，右边 n2+1\cfrac{n}{2}+12n​+1 个区域可以放 f(n2+1)f(\cfrac{n}{2}+1)f(2n​+1) 块能量石，那么去掉中间那个被重复计算的能量石之后，此时 nnn 个区域可以放置的能量石个数为 $$f(n) = f(\cfrac{n}{2}) + f(\cfrac{n}{2}+1) - 1$$

递归终止条件为 n=3n = 3n=3 或 n=4n = 4n=4 时，此时两边放置能量石之后中间都无法放置能量石，f(n)=2f(n)=2f(n)=2。记得 n=1n = 1n=1 和 n=2n = 2n=2 两个特殊情况，此时 f(n)=1f(n) = 1f(n)=1。

需要强调的是，当 nnn 为奇数时，千万不能把递归关系式写成 f((n + 1) / 2) + f((n + 1) / 2) - 1，这样每次 nnn 为奇数时都会比原来多调用自身一次，看起来只是多了一次，可是对于程序效率的降低是特别显著的，会导致 TLE。

计算出 f(m)f(m)f(m) 和 f(n)f(n)f(n) 之后，求能量石的总数，有生机区域的总数和无生机区域的总数就很简单了，详见示例代码，注意结果可能会爆 int，需要使用 long long。

示例代码

{

    else if (n <= 4)
        return 2;

int f(int n)
{
    if (n <= 2)
        return 1;
    int k = 0;

J 博丽灵梦的大清洗

考点 难度

题目分析

本问题被称为Tantalizer Problem，方法很多，例如维护等差数列，计算递推公式等。感兴趣的同学可以自行搜索调研

本题解采用递归的做法，将大的目标转化为更小且更易解决的子目标。

放到这道题来，就是将一个从 111 到 nnn 的序列不断转化为更小的从 111 到 nnn 的序列来处理。

题目中有两种操作：从上到下消去和从下到上消去，两种操作轮流执行，得到最终剩下的数字。

记 f(n)f(n)f(n) 为第一步从上到下消去时最后剩下的数字， f′(n)f'(n)f′(n) 为第一步从下到上消去时最后剩下的数字。由对称性可知 f(n)+f′(n)=n+1f(n)+f'(n)=n+1f(n)+f′(n)=n+1

随后我们考察从上到下进行消去时的情况。此时序列中所有的奇数被消去，只剩下偶数，且他们的值正好是 111 到 ⌊n2⌋\left\lfloor\dfrac{n}{2}\right\rfloor⌊2n​⌋ 的序列的两倍。由此可以得到 f(n)=2∗f′(⌊n2⌋)f(n)=2*f'(\left\lfloor\dfrac{n}{2}\right\rfloor)f(n)=2∗f′(⌊2n​⌋)

由题意可知初始条件为 f(1)=f′(1)=1f(1)=f'(1)=1f(1)=f′(1)=1

由上可以计算得到：

f(n)={1n=12∗(⌊n2⌋+1−f(⌊n2⌋))n>1f(n)=\begin{cases}1&n=1\\ 2*(\left\lfloor\dfrac{n}{2}\right\rfloor+1-f(\left\lfloor\dfrac{n}{2}\right\rfloor))&n>1\end{cases} f(n)={12∗(⌊2n​⌋+1−f(⌊2n​⌋))​n=1n>1​

由此，我们可以得到递归函数

int GreatPurge(int n) {
    if (n == 1) {
        return 1; // 初始条件
    } else {
        return 2 * (n / 2 + 1 - GreatPurge(n / 2)); // 递归调用函数
    }
}
